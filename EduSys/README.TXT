EduSys - Yêu cầu hệ thống

- Phiên bản JDK-17
- Phiên bản My SQL 7 trở lên
- JRE 1.8.0 Update 351

Đầu tiên, các bạn cần chạy file database EduSys và import dữ liệu vào

Tiếp theo, chạy file setup.exe để sử dụng ứng dụng

Nếu bị lỗi hãy thử chạy các file sau:

- Mở file: File sửa hệ thống
- Chạy 3 file lần lượt để ứng dụng có thể chạy trên hệ thống đó
